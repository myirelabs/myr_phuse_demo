# Basic MYR webpy DRO blockchain demo

DROs are reproducible computing units.

This is a basic web2py site showing the utility of blockchain for maintaining integrity of the research process using DROs

A copy of web2py is included for simplicity.

The application code is self contained in web2py/applications/myr

## Install

1. git pull $REPO_URL
2. cd webpy
3. python web2py.py
